# 医師採用面接チェックリスト

## 概要
- **目的**: 技術スキルより、人間性・チーム適性・コミュニケーション能力を優先評価
- **面接構成**: チーフ2人との面接 → 院長との面接
- **所要時間**: 合計2〜3時間（充分な評価時間を確保）

---

## 第1段階：チーフ医師面接（合計60〜90分）

### A. 基本情報確認（10分）
- [ ] キャリア経歴の概要確認
- [ ] 前職の離職理由を詳しく聞く
- [ ] 当院への応募動機

### B. 人間性・価値観の評価（20分）

**質問例：**
- 「前の職場での最も成功した経験は何ですか？」（協働能力、チームワーク）
- 「医師として大切にしていることは何ですか？」（価値観、患者観）
- 「ストレスを感じるのはどんな時ですか？どう対処していますか？」（メンタルヘルス、対処能力）
- 「チームメンバーとの意見が対立した時、どう対応しますか？」（柔軟性、協調性）

**評価項目：**
- [ ] 誠実さ・正直さ（虚偽がないか）
- [ ] 自己認識（強み・弱みを理解しているか）
- [ ] 医療人としての倫理観
- [ ] 他者への共感能力
- [ ] 学習意欲・成長マインド

### C. コミュニケーション・スキル評価（15分）

**質問例：**
- 「難しい患者さんや家族にどう対応していますか？」（説明能力、忍耐力）
- 「前の職場の同僚について、どう思いますか？」（他者への見方、人間関係構築能力）
- 「我々のスタッフとうまくやっていけそうですか？理由は？」（適性判断）

**評価項目：**
- [ ] 聞く力（質問に真摯に向き合うか）
- [ ] 説明の明確さ・論理性
- [ ] 非言語コミュニケーション（態度、アイコンタクト）
- [ ] 柔軟性・適応性
- [ ] チーム志向性

### D. 院内文化・チーム適性（10分）

**質問例：**
- 「理想的な医療チームはどんな組織だと思いますか？」
- 「新しい環境に入る時、何を大切にしていますか？」
- 「同じ職場の医師や看護師と意見が違う場合、どう調整しますか？」

**評価項目：**
- [ ] 当院のカルチャーへの親和性
- [ ] 下の職階のスタッフへの接し方（謙虚さ）
- [ ] 周囲への配慮・気配り

### E. 逆質問対応（5分）
- [ ] 候補者からの質問に対する受け答え
- [ ] 質問内容から見える関心事・価値観

---

## 第2段階：チーフ間でのフィードバック（10分）
- [ ] 全体的な印象の共有
- [ ] 懸念点がないか確認
- [ ] 次段階へ進むか判定

---

## 第3段階：院長面接（30〜45分）

### A. チーフからのフィードバック反映（5分）
- チーフの評価をベースに面接

### B. 医師としての責任感・患者観（15分）

**質問例：**
- 「医師としてのキャリアビジョンは？」（長期的な関心）
- 「患者さんにとって最良の医療とは？」（患者中心性）
- 「医療の失敗や課題にどう向き合っていますか？」（責任感、誠実さ）
- 「倫理的に難しい判断を迫られた経験はありますか？」（倫理観）

**評価項目：**
- [ ] 患者利益を最優先にしているか
- [ ] 医療の継続学習・向上への意欲
- [ ] 組織への貢献意思

### C. リーダーシップ・判断力（10分）

**質問例：**
- 「前職で主導した取り組みはありますか？」（主体性）
- 「簡単でない判断を下す時、何を重視しますか？」（判断基準）
- 「スタッフと意見が異なる時、最終的にどう判断しますか？」（責任感と柔軟性）

**評価項目：**
- [ ] 決断力・判断力
- [ ] スタッフへのサポート体制の考え
- [ ] 問題解決能力

### D. 技術的な側面（10分）

**質問例：**
- 「得意な分野・症例は？」
- 「苦手な分野での対応は？」（謙虚さ）
- 「最新の医療情報は何で得ていますか？」（学習姿勢）

**評価項目：**
- [ ] 技術レベルの確認
- [ ] 継続学習の姿勢
- [ ] 弱点への向き合い方

### E. 報酬・労働条件の確認（5分）
- [ ] 給与期待値
- [ ] 勤務時間・条件での懸念
- [ ] その他の質問

## 面接官向けのコツ

### 聞き方の工夫
- **開放質問を使う**：「〜ですね」ではなく「〜でしたか？」と聞く
- **沈黙を恐れない**：答えるまで待つ（本音が出やすい）
- **具体例を求める**：「それはどんな時ですか？」と掘り下げる
- **後付けの説明に注意**：直感的な答えの方が本音

### 見るべき非言語情報
- [ ] 目線・アイコンタクト（誠実性）
- [ ] 身体の向き（こちらへの関心）
- [ ] 手足の動き（落ち着き、不安感）
- [ ] 声のトーン（自信、ストレス）
- [ ] 笑顔の自然さ

### 避けるべき言動
- ❌ 一問一答で終わらせる
- ❌ 面接官同士で先に結論を出す
- ❌ 学位や資格だけで判断する
- ❌ 自分の価値観を押し付ける
- ❌ ネガティブな質問ばかり

---

## 面接後のプロセス

1. **チーフと院長で別々に評価**（会う前に）
2. **全員で協議**（結果を統合）
3. **一致しない場合は再度面接** または 詳しい理由を聞く
4. **最終判定**：採用・検討・不採用を決定
5. **フィードバック**：理由を丁寧に説明（採用・不採用とも）

---

## 補足：複数回面接のメリット

✅ **チーフ面接の利点**
- 日常的に一緒に働く医師たちとの相性を確認
- スタッフ観点での評価が重要
- 複数の目による多面的な評価

✅ **院長面接の利点**
- 医師としての倫理観・ビジョンを深掘り
- 組織戦略との合致確認
- 最終判定の責任者としての確認
