# ステラ返信下書きアシスタント（Chrome拡張）

LINE公式アカウントマネージャー上で、メモを書くだけで本人の文体に清書してくれる Chrome / Edge 拡張機能です。

## できること

- **返信欄に書いたメモを清書**：`6・11 10時半` → 「次回ご予約は6月11日10時半でお取りしました…」など、本人の文体で自然な返信文に展開
- **入力欄の右上に ✨ ボタンを常時表示**：押すだけでその場で清書 → 返信欄を置き換え
- **🚀 一括学習**：チャット一覧を自動で巡回し、最大1000件のメッセージから本人の文体を学習
- **📚 単発学習**：いま開いているチャットだけを学習サンプルとして追加
- 学習サンプルは Chrome のローカルストレージに保存（外部送信なし）

## インストール手順

### 1. ファイルを保存

#### 方法A: リポジトリ全体をZipダウンロード
1. GitHub のリポジトリトップで右上の緑色 **「Code」** ボタン
2. **「Download ZIP」** をクリック
3. ZIP を解凍し、`chrome-extension/` フォルダの場所を確認

#### 方法B: 個別ファイルを保存
- `manifest.json` / `content.js` / `content.css` / `popup.html` / `popup.js`

### 2. Chrome に拡張機能をインストール

1. `chrome://extensions` を開く
2. 右上 **「デベロッパーモード」** を ON
3. 左上 **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. 上記の `chrome-extension` フォルダを選択
5. 拡張機能一覧に「ステラ返信下書きアシスタント」が追加される

### 3. APIキー設定

1. ブラウザ右上の拡張機能アイコン 🧩 → 「ステラ返信下書きアシスタント」
2. Anthropic API キー（`sk-ant-...`）を貼り付けて「保存」

## 使い方

### A. まずは一括学習で文体を仕込む（初回1回だけ）

1. https://manager.line.biz/ を開いてチャット一覧を表示
2. 画面右上のパネルで **「🚀 一括学習 (最大1000件)」** をクリック
3. 拡張機能が自動でチャット一覧を順番にクリックし、各会話を学習サンプルに追加
4. 進捗バー（`学習中: 12チャット / 480メッセージ`）が動く間は触らずお待ちください
5. 最大1000メッセージ収集すると自動停止。途中で止めたい場合は **🛑 中止**

> 一括学習がうまく動かない場合（チャット一覧の構造を検出できない場合）は、各チャットを手で開いてから **「📚 この会話を学習」** で1件ずつ蓄積してください。

### B. 日常運用：メモ → 清書

1. 返信したいチャットを開く
2. LINE Manager の **返信欄にメモを書く**
   - 例: `6・11 10時半`（次回予約）
   - 例: `キャンセル可`
   - 例: `変更可 いつでも`
   - 例: `了解`
3. 入力欄の右上にある丸い **✨ ボタン** をクリック
4. 数秒待つと、返信欄が本人の文体で書かれた丁寧な文章に **置き換わる**
5. 内容を確認・必要なら微修正 → LINE Manager から送信

### C. 手動操作（メモを自分で打って結果を確認したい場合）

サイドパネルの「✏️ 手動でメモから生成する」を開いて、会話読み取り → メモ入力 → 生成 → 挿入の流れで操作できます。

## トラブルシューティング

### ✨ ボタンが入力欄の横に出ない
- ページを再読み込み（F5）
- チャットを開いて入力欄が表示されている状態か確認

### 「返信欄にメモを書いてから ✨ を押してください」と出る
- 返信欄が空のときは清書する元ネタがありません。先にメモ（短いキーワードでOK）を書いてください。

### 一括学習で「チャット一覧が見つかりません」と出る
- LINE Manager の UI 構造を自動検出できなかった可能性があります
- 個別チャットで「📚 この会話を学習」を 10〜20件 繰り返すフォールバックをご利用ください

### 学習しても文体が真似されない
- 学習サンプルが少ない可能性。パネル右上の `📚 N` を確認（5件以上推奨）
- 「本人/相手」判定は「右寄せ＝本人」のヒューリスティック。UI が変わると誤判定する場合あり

### API エラー
- APIキーが正しいか（拡張機能ポップアップで確認）
- Anthropic Console で残高があるか

## 注意事項

- 生成された下書きは **必ず内容を確認してから送信** してください
- メモに無い事実（別日時・追加情報）は AI が勝手に作らないようプロンプトしていますが、念のためチェック
- 患者さんの個人情報を含む会話を Claude API に送る点については、業務利用前にご確認ください
- Anthropic API は従量課金です（1回の清書で約 1〜3円、一括学習は保存のみで無料）
