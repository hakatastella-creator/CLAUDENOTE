# ステラ返信下書きアシスタント（Chrome拡張）

LINE公式アカウントマネージャー上で、AIによる返信下書きを生成・挿入できる Chrome / Edge 拡張機能です。

## できること

- LINE Official Account Manager または chat.line.biz の **入力欄の右上に ✨ ボタン** を常時表示
- ✨ ボタンを押すだけで、表示中の会話を読み取って Claude AI が下書きを生成
- 画面右上のサイドパネルから手動操作も可能
- **📚 この会話を学習**：複数の会話を学習サンプルとして蓄積し、本人の文体を真似た下書きを生成
- ワンクリックで入力欄に挿入、またはクリップボードにコピー

## インストール手順（5分）

### 1. このフォルダをダウンロード

#### 方法A: リポジトリ全体をZipダウンロード
1. GitHub のリポジトリトップで右上の緑色 **「Code」** ボタンをクリック
2. **「Download ZIP」** をクリック
3. ダウンロードした ZIP を解凍
4. `CLAUDENOTE-main/chrome-extension/` フォルダの場所を確認

#### 方法B: chrome-extension フォルダだけ手動でコピー
1. このフォルダ内の以下のファイルを、PCの任意の場所に保存：
   - `manifest.json`
   - `content.js`
   - `content.css`
   - `popup.html`
   - `popup.js`
   - `README.md`（任意）

### 2. Chrome に拡張機能としてインストール

1. Chrome を開く
2. アドレスバーに以下を貼り付けて Enter：
   ```
   chrome://extensions
   ```
3. 右上の **「デベロッパーモード」** トグルを **ON**
4. 左上に表示される **「パッケージ化されていない拡張機能を読み込む」** をクリック
5. ステップ 1 で保存した `chrome-extension` フォルダを選択
6. 拡張機能一覧に「ステラ返信下書きアシスタント」が追加される

### 3. APIキーを設定

1. ブラウザ右上の拡張機能アイコン（パズルピース 🧩）をクリック
2. **「ステラ返信下書きアシスタント」** をクリック（ピン留めしておくと便利）
3. ポップアップが開く
4. **Anthropic API キー** 欄に、`sk-ant-xxx...` で始まるキーを貼り付け
5. **「保存」** をクリック → 「保存しました ✓」が出ればOK

> APIキーは Chrome のローカルストレージに保存され、外部には送信されません。

## 使い方

### A. まずは文体を学習させる（推奨：最初に5〜10件）

1. **https://manager.line.biz/** （または chat.line.biz）にアクセス
2. **本人がいつも返信している会話** を1件開く（過去のやり取りが見える状態）
3. 画面右上のパネルにある **「📚 この会話を学習」** をクリック
4. パネル右上のバッジが `📚 1` に増えればOK
5. 別の会話を開き、同様に **「📚 この会話を学習」**
6. これを **5〜10件** 繰り返す（多様な相手・トーンの会話を混ぜると精度が上がります）

> 💡 学習サンプルは Chrome のローカルストレージに保存されます。拡張機能アイコン → 設定ポップアップで件数確認・全削除ができます。

### B. 下書きを生成する（学習後の通常運用）

1. 返信したい会話を開く
2. **入力欄の右上に出る丸い ✨ ボタン** をクリック
3. 自動で会話を読み取り、学習済みの文体で下書きを生成（3〜10秒）
4. 画面右上のパネルに下書きが表示される
5. **「▶ 入力欄に挿入」** で返信欄へ直接入力 or **「📋 コピー」**
6. 必要に応じて修正 → LINE Manager から送信

## トラブルシューティング

### 「画面から会話を読み取る」がうまく動かない
LINE Manager の UI が変わった可能性があります。手動で過去のやり取りと受信メッセージを貼り付けてご利用ください。

### ✨ ボタンが入力欄の横に出ない
- ページを再読み込み（F5）してください
- LINE Manager のチャット画面（入力欄が見えている状態）に入っていますか？
- 入力欄が見つからない場合は表示されない仕様です。サイドパネルから手動で操作してください。

### 「📚 この会話を学習」したのに文体が真似されない
- サンプル件数が少ない可能性があります。**5件以上** 蓄積してください
- 自動の本人/相手の判定は「右寄せ＝本人」のヒューリスティックです。UI が変わると誤判定する場合があります

### 「入力欄に挿入」がうまく動かない
LINE Manager 側の入力欄に直接コピペしてください。

### API エラーが出る
- APIキーが正しく設定されているか（拡張機能のポップアップで再確認）
- Anthropic Console で残高があるか
- APIキーが有効か（無効化されていないか）

### パネルが表示されない
- 拡張機能が有効になっているか（chrome://extensions で確認）
- ページを再読み込み（F5）
- LINE Manager のURL（manager.line.biz または chat.line.biz）で開いているか

## 注意事項

- 生成された下書きは**必ず内容を確認してから送信**してください
- 確定していない事実（日時・金額・約束）は AI が勝手に作らないようプロンプトしていますが、念のためチェック
- 患者さんの個人情報を含む会話を Claude API に送る点については、業務利用の前にご確認ください
- Anthropic API は従量課金です（1回の下書きで約 1〜3円）
