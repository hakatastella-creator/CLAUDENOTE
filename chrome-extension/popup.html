📚 文体学習サンプル

現在 0 件 蓄積中
LINE画面右上のパネルで「📚 この会話を学習」を押すと、その会話を本人の文体サンプルとして蓄積します。
5〜10件 蓄積すると、生成される下書きが本人の文体に近づきます。
使い方
1. LINE Official Account Manager / chat.line.biz を開く
2. 入力欄の右上に出る ✨ ボタンで即生成
　 または、画面右上のパネルから手動で操作
3. 文体を学ばせたい会話を開いて「📚 この会話を学習」をクリック
4. 「▶ 入力欄に挿入」または「📋 コピー」で送信
