# LINE返信漏れ検出（Chrome拡張機能）

LINE公式アカウントマネージャー (`manager.line.biz`) と `chat.line.biz` の
画面右上に **「📩 返信漏れ (N)」** バッジを常時表示する Chrome 拡張機能です。

クリックすると、未読バッジが付いているトーク（= 相手の発言で止まっていて、
まだ自分が返信していないトーク）の一覧をドロップダウンで表示します。
一覧の各行をクリックすると、該当トークへジャンプします。

AI下書きや外部APIは一切使いません。完全にローカル動作です。

## インストール手順

1. Chromeで `chrome://extensions/` を開く
2. 右上の「**デベロッパーモード**」をオンにする
3. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
4. このフォルダ（`chrome-extension-detector/`）を選択
5. LINE Manager / chat.line.biz のタブを **ハードリロード**（Ctrl + Shift + R / Cmd + Shift + R）

## 使い方

- 画面右上に赤い「📩 返信漏れ (N)」バッジが常時表示されます
- N の数字 = 未返信トーク数（15 秒ごとに自動更新）
- バッジをクリック → 未返信トークの一覧パネルが開く
- 一覧の行をクリック → そのトークを開く
- 🔄 で手動再取得、✕ で閉じる

## 検出ロジック

LINE 側の DOM は変わりやすいので、以下のヒューリスティックで検出しています:

1. トーク行候補を複数のセレクタで収集
   （`[class*="ChatList"] li`, `[class*="TalkList"] li`, `a[href*="/chat/"]` など）
2. 各行内に「未読バッジっぽい要素」があるかチェック
   （class名に `badge` / `unread` / `count` を含み、テキストが 1〜3 桁の数字）
3. 該当するものを「未返信」とみなしてリスト化

そのため、LINE 側で未読件数が消えると（既読扱いされると）リストからも消えます。

## 検出が効かないとき

LINE の UI 更新でセレクタが合わなくなった可能性があります。
DevTools でトーク一覧の HTML 構造（特に未読件数バッジの class 名）を
共有してもらえれば、`content.js` の `findUnrepliedTalks()` を更新します。
